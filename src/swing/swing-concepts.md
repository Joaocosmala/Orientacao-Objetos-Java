# Java Swing
Java Swing é uma biblioteca da Java Foundation Classes (JFC) usada para criar interfaces gráficas (GUI) em Java.
Ela é mais moderna e flexível que o AWT, oferecendo componentes como botões, caixas de texto, tabelas, menus e muito mais.

----------------------

# Os principais componentes do Swing incluem:
## JFrame:
Representa a janela principal de uma aplicação.

## Panel:
Um contêiner usado para organizar outros componentes.

## JButton:
Um botão que executa ações ao ser clicado.

## JLabel:
Exibe texto ou imagens.

## JTextField:
Permite a entrada de texto em uma única linha.

## JTextArea:
Permite a entrada de texto em múltiplas linhas.

## JComboBox:
Cria listas suspensas para seleção de itens.

## JOptionPane:
Exibe caixas de diálogo para mensagens ou interações.
